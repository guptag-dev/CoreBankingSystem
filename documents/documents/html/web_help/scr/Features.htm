Features  

  1. Includes different services as e-Banking , Bill Payment , Tax Payment , etc.

  2. Inbox Facility is also provided.

  3. User can Submit any Query to get the corresponding solution.

  4. Includes Auto Reports Generation on daily,monthly and yearly basis.

  5. Provision of Standing Instruction Issue.

  6. A search functionality is also included to any particular account details.

  7. It provides a section of Remittance which includes Inter- Banking as well as option of Foreign Remittances.

  8. Letters of Guarantee can also be issued.

  9. Incudes General Ledger Maintainence.

  10. Intra-Banking Facility can be used for account to account transactions.

  11. A Command Box is also provided which can be used to perform an operation directly by command-line.

  12. A SQL query can also be given to operate any function even those which are not included in the software. Administrator has full rights over the database while employee has limited access to it.
